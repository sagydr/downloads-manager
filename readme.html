Downloads Manager
This is the best extension to organize your downloads into different folders.

This is not an easy task. You need to understand one thing:
Due to security reasons Javascript doesn't allow access to ANY folder on your computer. So my suggestions are either:
    1. Set your default download location to be your main folder, usually c:\ or /Users/<username>/ (for mac) by changing the value in the (Advanced -> Downloads):
  1. Click the Chrome menu on the browser toolbar and select Settings  ( or click)
  2. Click Show advanced settings and scroll down to the "Downloads" section.
  3. Click Change and select your main folder.
    2. Place all your required download folders as sub-diretories of the main download folder.

Set Rules:

  1. Click the New Rule button



  2. Select Filename Rule Type and/or Extension Rule Type
    • none - don't apply any rule
    • exact match - input must match exactly the filename or extension.
    • part of - input needs to be a part of the filename or extension.
      e.g. "tor" is part of the extension .torrent.
      "science" is part of the filename "my_science_book".
    • regular expression - advanced usage. If you have knowledge of regular expressions you can define them here and they will be applied to the filename or extension.
      e.g ".*tor.*[0-9].*" is a valid regular expression input.

  3. Fill the Input box  


  4. Fill the Folder box


  5. Save


Examples

  1. save all mp4 files containing the word dexter into videos/tv/dexter folder.

  2. save all torrent files into downloads/torrents folder.

  3. save all files containing the word McDonalds into MyNewsLetters folder.

  4. save all files containing either word mp4, mov, mkv, avi into videos/general folder.

Q & A

Q: Are rules case-sensitive?
A: NO! All rules are case-INsensitive.

Q: How is "any video format" defined?
A: It is a regular expression combining all video file formats ("...m4v|mkv|mod|mov|mp4|mpeg|...") taken from: online-convert.com
    All other pre-defined formats are taken from there too.
