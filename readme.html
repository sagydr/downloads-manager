Hello. Tired of maneuvering between the different download folders? (torrents, pictures, movies, ...)
This is the best extension to organize your downloads into folders.

This is not an easy task. Due to security reasons you need to assist with the following:
Set your default download location to be your main folder, usually c:\ or /Users/<username>/ (for mac) by changing the value in the (Advanced -> Downloads):
  1. Click the Chrome menu on the browser toolbar and select Settings  ( or click)
  2. Click Show advanced settings and scroll down to the "Downloads" section.
  3. Click Change and select your main folder.

Set Rules:
  1. Click the New Rule button



  2. Select Rule Type
